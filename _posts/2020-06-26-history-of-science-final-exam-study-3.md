### 19세기 이전 생리학상의 변혁의 성격

천문학 상의 혁명이나 역학상의 혁명에 비해 내용이나 영향 면에서 소규모
- 피의 순환과 인체 시스템에 대한 사실적인 내용만 변혁되고, 전체적으로는 아리스토텔레스 체계 내에서 남아 있었다.
- 생물학이 아리스토텔레스 체계를 완전히 탈피하는 것은 진화론과 유전학이 나타나기 시작한 19세기에 이르러서야 가능하게 되었음.

## 자연발생설과 생물속생설

- 자연발생설 : "생명체가 부모 없이 스스로 생길 수 있다" 는 가설
    - Aristoteles 가 주장한 생명의 기원에 관련된 학설 중 하나 <- 목적론적 세계관에 기초
    - 자연발생설의 견해로 18C 까지 미생물학의 발전은 거의 없었음.

- 생물속생설 : "생물체는 다른 생물체로부터만 발생할 수 있다" 는 입장

<br>

### 자연발생설 부정의 역사

자연발생설이 부정되는 과정에서 실험세균학 대두, 멸균의 개념 등장, 식품 보관 방법 발전함

- Francesco Redi : 두 병에 죽은 물고기를 넣음. 한쪽 병은 뚜껑을 덮고, 다른 병은 잘 짜여진 천으로 싸서 막음. 며칠 방지함. 뚜껑을 덮지 않은 병 쪽에는 날벌레가 꼬임, 뚜껑을 한 병에는 날벌레가 꼬이지 않음.
-> 과학에서 기본적인 대조의 개념 확립
-> 반론 : 날벌레나 파리 등에 관해서만 자연발생이 부정된 것 뿐

- Redi 의 실험을 반박하기 위한 J. T. Needham 의 실험
    - 고기 국물을 가열해 시험관에 넣고 마개를 막은 실험에서 미생물이 존재
    - 오류 : 마개가 잘 닫기지 않은 점. 저온 가열.

- Lazzaro Spallanzani : 멸균 개념을 만들어 자연발생설 부정
    - 용기에 유기물 용액을 고온 가열 후 밀폐 => 장기간 보존해도 미생물은 발생 못함.
    - 식품의 보관방법에 큰 영향을 줌.
    - 반론 : 산소가 공급되지 않았기 때문에 자연 발생한 미생물의 생육이 억제된 것 뿐.

- Louis Pasteur : 세균학의 아버지 / 세균 병인설 증명 / 저온 살균법 / 자연발생설 비판 : 발효는 미생물의 증식 때문
    - 백조목 플라스크 실험으로 자연발생설 부정
        - 부패성 건초 추출물 구형플라스크에 담고 가열 => 세균 발생하지 않음
        - 반론 : 가열과정에서 열이 공기의 발생능력 빼앗음
        - 오염되지 않은 장소(지하실, 높은 산 위)에서 플라스크 뚜껑 열어놓은 상태로 실험

<br>

### 자연발생설 재등장

- Aleksandra Ivanovish Oparin : The Origin of Life on Earth 에서 원시상태의 지구에서 무기 물질로부터 유기물 및 생명이 발생하는 과정을 과학적으로 설명-주장 <- 기계론에 기초

- Urey 와 Miller : 실험으로 오파린의 이론 뒷받침

<br>

### 전성설과 후성설의 논쟁

- 전성설 : 수정란이 발생하여 성체가 되는 과정에서 개개의 형태와 구조가 이미 알 속에 갖추어져 있어 발생하게 될 때 전개된다 정자와 난자 안에 성체의 축소형이 존재
    - 모든 기관들은 발생과정을 거치는 것이 아니라 성장하는 것으로 생각

- 후성설 : 생물이 발생하 때 단순한 상태에서 복잡한 상태로 발전하여 구조가 새롭게 생겨난다. 독일의 생물학자 Wolff 가 주창함.

#### 전성설의 부정

- Caspar Friedrich wolff : 닭의 발생을 현미경으로 관찰<br>
=> 어떤 기관도 성장하는 것이 아니라 배의 발생 도중에 하나씩 형성됨을 밝힘.

- Karl Ernst von Baer : 발생과정에서 일반형질이 특수형질보다 먼저 나타난다.

- Ernst Haeckel : 개체 발생은 개통발생을 반복한다.

발생설에 대한 논쟁은 세포설이 체계화되고 체세포와 생식세포의 염색체 수가 다르다는 것이 밝혀지면서 종지부를 찍게됨.

<br>

## 멘델의 유전법칙

Gregor Johann Mendel : 수도회 신부. 유전학의 창시자. 유전학의 수학적 토대 마련.

- 오스트리아제국 메렌지방의 소작농가에서 출생. 유년시적 농사-원예 일 도움.
- 아우그스티노회 입회, 사제서품, 수도원장 추천으로 빈 대학교에서 자연과학 공부, 브린종합학교 교사
- 1856년 교회 뜰에 완두 실험 재배 시작, 7년 후 '멘델의 법칙' 발견.
- 식물 잡종에 관한 연구 발표 => 주목을 받지 못함. 수도원장에 선출. 동료 수도사들과 그가 살던 지역의 사랑과 존경을 받았으나, 당시의 유명한 생물학자들에게 전혀 알려지지 않았음.
- 1900년 코렌스, 드 브리스 등에 의해 재조명되고 사람들의 주목을 받게 됨.
- 근대과학의 방법론을 충실히 따른 생물학 사상 뛰어난 실험 연구
    - 관찰 -> 가설 설정 -> 실험 -> 법칙 수립
    - 계획의 교묘함, 실험의 정확성, 탁월한 자료 처리법

<br>

## 생명현상 이해의 접근방법 변화

- 생기론적 관점에서 기계론 관점으로 
    - 유기체 전체로서 통합적 이해의 관점에서 세포와 같은 미시적 대상의 생명현상의 이해를 기초로 생물체 전체로서 생명현상 이해로
        - 환원적 기계론, 환원적 물리주의<br>
        <= 현미경, 과학 기기의 발달, 화학의 발전
        - 세포의 구조, 세포의 기능, 물질대사 메커니즘 (20C)

    - 1930 년대 등장한 유기체론(Organicism)
        - 생기론과 환원적 물리주의 거부
        - 비환원적 기계론, 비환원적 물리주의 : 환원주의를 보완하는 수준

```
 생기론적 관점   =>   기계론적 관점
(목적론적 관점)     - 환원적 기계론 : 미시적
    거시적          - 비환원적 기계론 : 전체론적
                      (유기체론적 관점)
```

<br>

### Vitalism, Mechanism and Organicism

- 생기론 : Aristoteles 에서 유래, 목적론을 내포함. 기계론이나 유기체론과 반대 입장.
    - 생명현상의 합목적성을 자연적 원인에 의해서는 설명할 수 없다는 입장<br>
    -> 물리-화학적 지식을 바탕으로 생명현상을 이해할 수 없음.
    - 생명체에만 존재하는 특별한 힘(Entelecheia)에 의해 생명현상이 나타남.
        - Entelecheia : 목적론적 완성태. 질료로부터 형상을 실현시키는 원리. 생명현상의 이면에 보이지 않는 힘.
    - 17C 등장한 생명기계론과 대립하여 여러가지 생기론이 나타남
    - 생명현상의 물리-화학적 특성이 밝혀짐에 따라 차츰 소멸됨.

- 기계론적 생명관 : 생명체가 섬세한 기계와 같다는 입장(생명기계론)
    - 생명 현상들도 물리-화학적 과정으로 환원해 설명할 수 있음. <- 환원적 유물론(기계론)
    - 혈액순환을 발견한 하비, 데카르트 주장

- 유기체론(Organicism) : 유기체적 자연관, Holism 과 동의어 <- 비환원적 유물론(기계론)
    - 우주 또는 그 부분들을 생명체와 같은 유기적 통일체들로 보는 세계관
    - 자연을 스스로 성장과 발생을 거듭하는 하나의 생명체처럼 파악.
    - 생명현상의 원리를 물리학-화학으로 환원해서 이해할 수 없다.
    - 현대 생물학에서 유기체의 자기조직화 특성을 강조하는 접근 방법의 이론
    - 생물학자 John Scott Haldane 이 이 용어를 처음 사용

## 진화론의 등장

### 진화론의 전개과정

- 19C 초 Lamarck : 무척추동물의 체계, 동물철학, 용불용설, 획득형질 유전
- Charles Darwin 의 종의 기원 : 자연선택, 획득형질 유전
- 바이즈만 : 체세포의 변화는 생식세포에 영향을 주지 못한다.
- 20C 초 돌연변이 발견 : 
    - 드 브리스 : 달맞이 꽃 교배 실험에서 멘델 법칙 재발견, 돌연변이 발견, 진화의 요인으로 돌연변이 주장
    - 모건 : 염색체 유전학 창시자, 초파리 염색체 연구로 돌연변이 증명<br>
    => 신 다윈주의
- Karl Pearson 주축의 생물통계학자들 그룹 : 계속적이고 개별적인 변이의 축적이 종을 형성
- J. D. Watson 과 H. Crick : DNA 이중나선 구조 발견
- 기무라 : 돌연변이의 우연적 집단 내 축적
- 지구과학의 발전 => 지리적 격리설

<br>

### 현대의 진화론

돌연변이, 유전자 단위의 작은 변화의 축적, 지역적 격리, 자연선택 등이 복합적으로 작용하여 진화가 일어남.

- Lamarck : 체계적인 학설
    - 가설 : 용불용설, 획득형질 유전
    - 획득형질은 유전한다는 가설을 증명하지 못함.

- 퀴비에 : 해부학 지식으로 화석 생물 재구성
    - 종의 불변론 주장 => 라마르크 이론 부정
    - 노아의 대홍수로 화석에 나타나는 종은 멸종됨.

<br>

### Charles Robert Darwin

- 에딘버러에서 의학 포기, 캠브리지에서 신학 전공
- 비글호에 박물학자로 승선
- Malthus 의 인구론에 영향을 받음
- 종의 기원
    - 진화의 메커니즘으로서 자연선택
    - Lamarck 와 같이 획득형질 유전 받아들임
    - 20C 초 돌연변이 발견으로 더욱 지지를 받음

<br>

비글호 여행에서 Darwin 의 관찰
- 화석으로만 보던 종들이 실제로 남미에서 발견
- 지역에 따라 같은 "종"들이 서로 차이가 남
- Galapagos 군도 여러 섬에서 서로 다른 동물군, 식물군이 존재.

Selection?
- Artificial Selection
    - 원예가-동물 사육가의 품종 개량
    - 자연에서는 이것이 어떻게 일어나느냐?

- Natural Selection
    - 자연에서는 다른 종들 간의 경쟁
    - Population (개체 수) 개념 중요<br>
    => 진화의 메커니즘

종의 기원이 발간되기까지
- Darwin 은 자연선택을 진화의 메커니즘으로 결론을 내림
- 종교적 저항 염려, J. D. Hooker 를 비롯한 측근에게만 알리고 20년이 넘도록 발표하지 않음
- A. R. Wallace 의 논문 도착
    - 공동발표 : 런던의 린네 학회
    - 큰 주목을 받디 못함
- 종의 기원 출간

Darwin 의 결론
- 화석으로만 보던 종들이 실제로 남미에서 발견
- 시간과 지역에 따른 종들이 차이 남.
- 기후와 풍토가 같음에도 다른 종들이 존재<br>
=> 기후-풍토-환경 등이 기계적으로 종의 분포를 결정하지 못함.
- 진화는 일어나고 외부 환경에 대한 적응이 진화에 중요하다.

Malthus 의 인구론
- 다윈이 재미 삼아 읽음
- 생존경쟁
- 같은 종들간의 경쟁
