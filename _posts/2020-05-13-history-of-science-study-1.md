<h1 id = "1">인간과 과학</h1>

<h2 id = "1.1">1.1 인류의 등장</h2>

<b>현존하는 인류는 사람 속(Genus) 안에서 단일 종(Species), 단일 아종(Subspecies)의 무리이다.

<br>

>
생물의 분류 단계는 스웨덴의 식물학자인 카를 폰 린네가 제시한 범위에 역(Domain)을 추가하여 분류한다.<br>
>
역(Domain) - 계(Kingdom) - 문(Phylum, 식물이면 Division) - 강(Class) - 목(Order) - 과(Family) - 속(Genus) - 종(Species)<br>
>
아종(Subspecies)은 서로 교배하여 생식능력이 있는 자손을 생산할 수 있지만 자연적으로 교배하지 않는 집단을 의미한다.<br>
인간의 경우 아종이 없기 때문에 단일 아종이다.<br>
>
과거에는 네안데르탈렌시스 등의 아종이 있었을 것이라 예상하지만, 모두 멸종하였다고 한다.<br>
>
그리고 인종이라는 개념은 생물학적으로 거의 의미가 없다고 한다.<br>
왜냐하면 플라이스토세 말에 인류는 거의 멸종 위기를 맞았고,<br>
소수만 살아남아 번식하여 현재에 이르렀는데 그 때문에<br>
유전자 풀이 좁아졌기 때문이다.<br>
>

<br>

<h3 id = "1.1.1"> 간략한 인류사 </h3>

인류의 진로를 이끈 세 가지 대혁명

- <b>인지 혁명</b>(7만 ~ 3만년 전) : 허구를 말할 수 있는 능력, 집단생활이 가능해짐<br> 
-> 이중의 실재 속에 살게 되었다. 객관적실체(강, 나무, 사자 같은 것) + 가상의 실재(신, 국가, 법인 같은 것)<br>
<b>=> 인류가 생물학적 자연의 존재에서 독립을 선언한 지점이다.

- <b>농업혁명</b>(1.2만년 전) : 구석기 시대의 채집 경제에서 신석기 시대의 생산경제로 전환<br>
-> 잉여 농산물 발생, 정착생활, 계급 발생(지배자와 엘리트, 국가체제 발생)<br>
<b>=> 농업혁명은 인류가 자연을 지배하기 시작하는 계기가 됨.

- <b>과학혁명<br>
-> 500여 년 간 인구가 5억 명에서 70억 명으로 늘어났다.<br>
-> 무지의 발견과 인정<br>
-> 교조적 독단의 거부 : 어떤 이론도 신성하지 않으며, 도전을 벗어난 대상이 아님.

<h2 id = "1.2"> 1.2 과학이란?</h2>

Science systematically builds and organizes knowledge<br>
in the form of testable explanations and predictions about the universe.<br>

<br>

>
과학은 실험가능한 가설과 우주에 대한 예측 안에서 체계적으로 지식을 만들고 구성하는 것이다.
>

<br>

<h3 id = "1.2.1"> 과학의 어원 </h3>

라틴어 scio- : 알다<br>
인도-유럽어 scio- : 분별하다, 구분하다<br>
중세 ~ 18세기 : 모든 종류의 체계적이거나 정확하게 기록된 지식<br>

과학은 철학이라는 단어의 넓은 의미로부터 구별되어 쓰이지 않는다.<br>
- 고대 ~ 근대의 철학은 자연철학, 윤리철학이었다.
- 박사 ==  Ph. D : Doctor of Philosophy 등등

<br>

<h3 id = "1.2.2"> 그래서 과학이란 </h3>

사물의 구조-성질-법칙에 대한 이론적 인식활동 및 그 산물로서 지식체계<br>

경험주의와 방법론적 자연주의에 근거하여 실험과 관찰을 통해 얻어낸 자연계에 대한 지식 체계<br>
<b>ㄴ좁은 의미의 과학</b><br>

- 경험주의 : 인식의 근원이 경험에 있다고 파악하는 이론
- 자연주의 : 실제의 사물과 현상을 자연 세계의 범위 안에서 파악하고자 함.
- 방법론적 자연주의 : 과학적 방법을 초월적 존재의 존재 여부를 추정하지 않는 자연적 설명으로 제한함.

<br>

<h3 id = "1.2.3"> 과학적 방법 </h3>

연역과 귀납의 논리를 바탕으로 한 이론과 재현성있는 실험을 바탕으로 한 과학

과학의 조건은 객관성과 재현성 이며, 엄밀한 연구방법 때문에 신뢰 받는다.<br>
실증주의, 긍정주의 : 과학의 방법을 인간 지식의 원천으로 인정하고 신뢰한다.

인식 발전의 3단계 : 신화적 단계 => 형이상학적 단계 => 실증적 단계<br>
(개인의 발전에도 적용된다.)

<br>

과학은 경험성, 객관성, 합리성을 갖는 보편적 지식이다.
- 경험성 : 탐구영역은 경험의 세계로 한정된다.
- 객관성 : 실험-관찰은 반복가능해야 한다.
- 합리성 : 현상은 세계 안에서 인간의 이성으로 설명이 되어야 한다.

<br>

>
반복가능해야 객관성을 유지할 수 있다.
>

<br>

<h3 id = "1.2.4"> 과학의 분야</h3>

<h4 id = "1.2.4.1"> 수학과 논리학은 과학인가? </h4>

- 엄밀한 방법을 통한 체계적 연구라는 점에서 다른 과학과 유사하다.
- 실험-관찰과 같은 검증으로 이루어지는 과학적 방법론이 사용되지 않아서 곤란하다는 견해도 있음.
- 과학에 매우 중요한 요소로 이용된다.

<br>

<h4 id = "1.2.4.2"> 경험과학 </h4>

- 과학의 방법론상 과학이다.
- 자연과학
- 응용과학 : 공학, 의학
- 사회과학
- 과학적 지식이란 <b>아직까지 반증되지 않은</b> 확고한 경험적 사실을 근거로 한 보편성과 객관성이 인정되는 지식의 체계이다.

<br>

<h4 id = "1.2.4.3"> 선험적 과학 - 형식과학 </h4>

경험적 사실이나 경험적 개념을 대상으로 하지 않는 과학<br>
경험과학과 대립되는 영역<br>
신학, 철학, 형식논리학, 수학<br>
그 중에 신학, 철학은 과학이라 할 수 없다.<br>
보편성이 인정되는 형식논리학이나 수학은 넓은 의미의 과학이다.

<h1 id = "2"> Appendix </h1>

<h3 id = "2.1.1">  논리학 그리고 연역과 귀납 </h3>

- 논리학 : 추론의 형식적 원리에 대한 학문<br>
- 추론 : 어떠한 판단을 근거로 삼아 다른 판단을 이끌어 내는 것.
    - 전제 : 이미 알려진 사실을 바탕으로 하는 조건으로서 명제
    - 결론 : 전제를 바탕으로 필연적으로 이끌어낸 새로운 명제

<br>

<h3 id = "2.1.2"> 연역, 연역 추론 </h3>

일반적인 것에서 특수한 것으로 나가는 추론 <br>
(일반적 원리를 전제로 하여 개별적 사실이나 특수한 다른 원리를 이끌어 내는 추론)

- 귀납 추론과 달리 명제들 간의 관계와 논리적인 타당성을 따진다.
- 연역 추론은 전제들로 부터 절대적인 필연성을 가진 결론을 이끌어 낸다.

- 직접 추론 : 한 개의 전제로부터 새로운 결론. ex) 대우명제
- 간접 추론 : 둘 이상의 전제로부터 새로운 결론. ex) 삼단논법

<br>

<h3 id = "2.1.3"> 귀납, 귀납 추론 </h3>

특수한 것에서 일반적인 것으로 나가는 추론<br>
부분에서 전체로 나가는 추론

특수한 사실들이나 현상들로부터 일반-보편 명제 및 법칙을 유도해 내는 간접 추론<br>
경험적 사실을 바탕으로 함.

연역과는 달리 사실적 지식을 확장해 줌.<br>
전제가 결론의 필연성을 논리적으로 확립해 주지 못함.

<br>

<h3 id = "2.1.4"> 연역 vs. 귀납 </h3>

논리학이 다루는 것은 연역이다.<br>
논리학은 추론에 있어서 내용이 아니고 형식을 다룬다.

|                           |       연역      |          귀납        |
|:-------------------------:|:---------------:|:--------------------:|
|         특수/일반         | 일반에서 특수로 |   특수에서 일반으로  |
|         경험/선험         |  선험성에 기초  | 경험적 사실들에 기초 |
|   세계에 대한 지식 확장   |      불가능     |          가능        |
| 추론의 결론에 대한 확신성 |      확신성     |          개연적      |
{: rules="groups"}

<br>

<h3 id = "2.1.5"> 고전 논리의 삼원칙 </h3>

Aristoteles 이후 전통적 논리학(형식논리학)의 근본원리

형식논리학 : 초시간적-형식적 관계를 문제삼는다.<br>
cf. 변증법적 논리학은 존재의 변화를 다룬다.

- 동일률 : 다른 것과 대립-구분되면서 변함없이 동등하게 존재하는 성질의 법칙
- 배중률 : 참이면서 동시에 거짓일 수 없다. A ∨ ~A
- 모순율 : 한 명제는 동시에 그것의 부정일 수 없다. ~(A ∧ ~A)

<br>

>
<h4 id = "2.1.5.1"> 배중률과 모순율의 구별법 </h4>
>
쉬운 모순부터 생각해보자.<br>
모순은 창과 방패 고사에서 나온 것을 알고 있을 것이다.<br>
우선 명제가 2개 이다.
```
    이 창은 모든 방패를 뚫을 수 있다.
    이 방패는 모든 창을 막을 수 있다.
```
서로 다른 명제가 상충되는 판단을 하는 것이다.
>
배중률은 참 거짓 외에 제 3의 판단을 허용하지 않는 것이다.<br>
(베 : 배제하다 중 : 중간을 )<br>
주로<br>
```
    이것은 A 이거나 A 가 아니다.
```
라는 문장이 대표적이다.
>
